# QR Code Generator Web App

A simple, responsive, and user-friendly QR Code Generator web application built using Flask and Bootstrap. This project allows users to generate QR codes from URLs or text and download them instantly.

---

##  Features

- Generate QR codes from any URL or text input
- Download generated QR codes instantly
- Responsive design (mobile, tablet, desktop)
- Clean and modern UI using Bootstrap 5
- Fast and lightweight Flask backend

---

##  Tech Stack

- Python
- Flask
- HTML5
- CSS3
- Bootstrap 5
